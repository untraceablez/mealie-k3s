# Welcome to Mealie-k3s

This repository provides a self-contained Helm chart for deploying the [Mealie](https://docs.mealie.io/) recipe manager on Kubernetes.

## Features

* **Mealie App**: The core application.
* **PostgreSQL Database**: A persistent database backend.
* **MetalLB**: Automatic `LoadBalancer` IP assignment.
